fun <T : Token> createSorted(tokens: Iterable<T>): TokenSequence<T>

Creates a new sequence of several tokens. The tokens are ordered by their positions (see Token.positions). If a token corresponds to multiple positions, it will occur multiple times in the sequence.

fun <I : Token, O : Token> createSorted(tokens: Iterable<I>, mapper: (I) -> O): TokenSequence<O>

Creates a new sequence from several tokens of type I that are mapped to a different type O. The tokens are ordered by their positions (see Token.positions). If a token corresponds to multiple positions, it will occur multiple times in the sequence.
