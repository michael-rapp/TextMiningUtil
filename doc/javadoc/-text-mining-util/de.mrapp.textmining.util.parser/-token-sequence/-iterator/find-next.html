findNext

@JvmOverloads fun findNext(matcherFunction: (TokenType) -> Boolean, maxSteps: Int = -1): Boolean

Searches for the next token that matches a specific matcherFunction. If such a token is found, true will be returned and calling next will result in the found token to be returned. If no matching token can be found, false will be returned and the iterator will point to the token that is located after the token, the search looked at last.

Parameters

maxSteps - The maximum steps to look ahead or -1, if the search should be continued until the end of the sequence is reached
