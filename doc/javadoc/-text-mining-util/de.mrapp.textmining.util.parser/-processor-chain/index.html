ProcessorChain

class ProcessorChain<I, O> : Processor<I, O>

A meta-processor that executes several processors in a predefined order. The result of the first processor is passed to the second processor and so on.

Parameters

I - The type of the data to be processed by the first processor

O - The type of the resulting data of the last processor
