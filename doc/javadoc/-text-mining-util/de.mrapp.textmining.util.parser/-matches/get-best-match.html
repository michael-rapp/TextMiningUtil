getBestMatch

@JvmOverloads fun getBestMatch(tieBreaker: TieBreaker<Match<F, S>>? = null): Match<F, S>?

Returns the best match according to the matches' heuristic values. Optionally, a tieBreaker can be specified. If two matches have the same heuristic value, it is used to decide which one of them is considered to be better.
