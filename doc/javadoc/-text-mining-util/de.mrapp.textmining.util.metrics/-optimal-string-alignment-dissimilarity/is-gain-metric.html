isGainMetric

val isGainMetric: Boolean

Overrides TextMetric.isGainMetric

True, if the metric is a gain metric, i.e. that two strings that evaluate to a greater heuristic value are considered to be more similar than two strings that evaluate to a smaller heuristic value, false otherwise.
