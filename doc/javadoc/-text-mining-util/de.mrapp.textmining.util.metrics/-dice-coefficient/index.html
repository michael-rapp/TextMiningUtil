DiceCoefficient

class DiceCoefficient : TextMetric

Allows to measure the similarity of texts according to the Dice coefficient. The dice coefficient measures the percentage of n-grams, which occur in both texts. It always evaluates to a heuristic value within the interval 0,1. Texts that reach a greater heuristic value are considered to be more similar than texts that reach a smaller heuristic value. The dice coefficient can also be applied to texts with different lengths.
