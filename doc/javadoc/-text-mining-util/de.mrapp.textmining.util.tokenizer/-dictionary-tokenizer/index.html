DictionaryTokenizer

class DictionaryTokenizer<K : CharSequence> : AbstractTokenizer<Substring>

Allows to split texts based on a Dictionary. E.g. given a dictionary that contains the entries "one" and "two", the text "abconedeftwo" is split into the substrings "abc", "one", "def", and "two".
