addToken

protected fun <T : Token> addToken(map: MutableMap<String, T>, token: String, position: Int, tokenFactory: (String, Int) -> T): Unit

Adds a new token to a map, if it is not already contained. Otherwise, the token's position is added to the existing token. A tokenFactory for converting the given token to the type T must be provided.

T - The type of the token
